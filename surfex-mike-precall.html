Step 6 -- Main one
Set up your second brain -- Surfex job and quote vault
The permanent layer for every quote, job, and customer record

FounderBud has two memory layers. SQLite holds recent conversations -- auto-purges after 3 days. Obsidian is the permanent layer: voice-to-quote transcripts, pipeline records, job phase notes, customer records, content drafts, invoice records.

🎙️ Voice-to-quote 📋 Quote pipeline 🔧 Job phase tracker
Every voice note Mike sends lands in inbox/. The bot transcribes it, extracts the job details, and routes it to the right folder -- quotes/draft/ if it's a new quote, jobs/active/ if the job is already booked. The quote pipeline tracks every open opportunity from first enquiry through to booked or dead. The multi-day phase tracker lives in jobs/active/ -- one file per job, phases: Prep, Primer, Base coat, Topcoat, Cure, with Mike and Callum assigned per phase.

Reference -- bot commands

Commands you'll use from day one

Type any of these in Telegram to control your bot. Or just talk -- commands are shortcuts.

/help
Lists all available commands
/status
Shows if the bot is running, which model, and memory stats
/model
Switch between Claude models -- faster for quick tasks, Opus for complex analysis
/voice
Toggle voice output on or off
/newchat
Fresh session. Memory persists; active context resets. Good for switching between quoting mode and job tracking.
/memory
Show what FounderBud knows about your setup -- Surfex, rates, job types, quote pipeline
/stop
Interrupt a running task
/dashboard
Get a link to the live web dashboard -- scheduled tasks, memory, conversation history
